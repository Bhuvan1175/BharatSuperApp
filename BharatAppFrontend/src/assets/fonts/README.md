# Fonts

Drop the following .ttf files here, then run `npx react-native-asset` to link them:

- Poppins-Regular.ttf, Poppins-Medium.ttf, Poppins-SemiBold.ttf, Poppins-Bold.ttf
- RobotoMono-Regular.ttf (for data annotations)

Until the fonts are linked, `src/theme/typography.ts` falls back to the system font so the app still runs.
